Audio Playlists
===============

* Audio playlists can be made using the same files used in the "File Based Game" section
* Likewise playlist can be given a menu using these safe files
* A more custom looking menu and controles can be used by creating custom HTML as described in the "Standard Games" section
* If you would like the player to be able to both control and listen to the audio playlist with the device they are holding, you should refer to the techiques layed out in the "Direct Games" section
* Audio Playlist can be made both for a game card or for the media center wich is stored on the microsd
